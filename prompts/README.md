# AI Helper Prompts

This folder contains prompts to help AI assistants (like Claude, ChatGPT, or Cursor) work effectively with this codebase.

## Available Prompts

| Prompt | Description |
|--------|-------------|
| [add-feature.md](add-feature.md) | Add a new feature to the app |
| [add-database-table.md](add-database-table.md) | Add a new database table with CRUD operations |
| [add-ipc-handler.md](add-ipc-handler.md) | Add a new IPC handler for main/renderer communication |
| [add-settings.md](add-settings.md) | Add a new settings option to the Settings panel |
| [debug-issue.md](debug-issue.md) | Debug and fix issues in the app |

## Usage

Copy the prompt content and paste it into your AI assistant, replacing the placeholders with your specific requirements.

## Example

```
I want to add a new feature: [User Profile Management]

<paste content from add-feature.md>
```
